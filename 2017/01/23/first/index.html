Unity3D学习之路—基本界面简介


　　刚开始新建一个Unity项目时界面和上图不一样，可以按习惯拖动各个窗口，如上图所示。上图Unity版本为3.5.6，后续版本都大同小异，下面介绍一下每个部分的功能。
　　Scene窗口：用于观察存放在世界的物体（GameObject）。开发者视角，开发者可以随意调整视角到想要观察的位置，鼠标+键盘wasd控制，也可以仅用鼠标控制。
　　Game窗口：游戏视角，即呈现在屏幕上玩家所看到的视角。新建场景时场景会有一个摄像机，摄像机视角即为玩家视角，开发者可以添加多个摄像机，可以控制摄像机的渲染顺序，后续文章会有摄像机的详细介绍。
　　Hierarchy窗口：显示所有物体（GameObject）以及层级关系，如上图所示已经有一个Main Camera物体，在该窗口双击物体（GameObject）可以在Scene窗口快速找到这个物体（GameObject）。
　　Project窗口：显示当前项目所有的文件，在该窗口点击鼠标右键可以新建一些文件夹或者文件，Unity有的资源需要再指定命名的文件夹内,后续文章会有详细介绍。
　　Inspector窗口：该窗口显示所选中的物体的属性，如果没有选中任何东西，该窗口为空。
　　
　　
　　
　　
　　上图为场景窗口上面的几个小按钮，作用于场景窗口。
　　1. 手型工具，用于平移整个场景的试图，不改变任何物体属性，快捷键Q；
　　2. 平移工具，用于平移选中的物体，快捷键W；
　　3. 旋转工具，用于旋转选中的物体，快捷键E；
　　4. 缩放工具，用于缩放选中的物体，快捷键R。
　　
　　
　　
　　
　　上图两个工具，左边点击选择物体的原点和中心，有时物体的原点和中心并不重合。右边的点击选择物体是相对local旋转还是相对global旋转。
　　
　　
　　
　　
　　上图三个按钮分别代表运行，暂停，和逐帧运行。
